USER INSTRUCTIONS
POWER ON:
To power on the unit, plug into an outlet.
CLOCK:
To set the clock. Hold the SELECT buon for 5 seconds. Use INCREASE/DECREASE buons to set the me on
the clock.
ROAST:
1. Add dry ingredients to ROAST CANISTER. Place oponal lter inside of ROAST INSULATOR. Place ROAST
INSULATOR on ROAST CANISTER.
2. Place ROAST CANISTER on HOTPLATE of HANABLEND appliance.
3. Use navigaon buons to highlight ROAST. Press SELECT or FORWARD buon to enter into ROAST mode
and advance to next step.
4. Set the mer by using the INCREASE and DECREASE buons. Press SELECT or FORWARD buon to
advance to next step.
5. Set the TEMP by using the INCREASE and DECREASE buons.
6. Press START. ROAST funcon will run.
7. When complete, allow to cool. Use CAUTION when handling hot ROAST CANISTER.
INFUSE:
1. Transfer dry ingredients to INFUSION BASKET and place INFUSION BASKET in INFUSION CANISTER.
2. Add desired volume of oil. Place INFUSION CANISTER on HOTPLATE.
3. Use navigaon buons to highlight and select INFUSE.
4. Set the mer by using the INCREASE and DECREASE buons.
5. Press FORWARD arrow to highlight CBD%.
6. Set CB% using INCREASE/DECREASE buons.
7. Press FORWARD or SELECT to highlight THC%.
8. Set TH% using INCREASE/DECREASE buons. Pressing SELECT will advance to next step.
9. Use navigaon buons to select WEIGH funcon. If readout is not 0.0g use the TARE funcon.
10. Use navigaon buons to highlight TARE. Press select buon. When readout is 0.0g, place dry
ingredients on scale. Press select to enter weight.
11. Add dry ingredients to basket and place in CANISTER. Use CAUTION if sll hot.
12. Use navigaon buons to highlight OIL. Use INCREASE/DECREASE buons to enter amount of oil used.
Oil can be measured and added to canister before or immediately aer this step.
13. Place LID on CANISTER. Place CANISTER on hotplate.
14. Use navigaon buons to highlight TIME. Use INCREASE/DECREASE buons to enter TIME. Press select
to enter me and advance to next step.
15. Use INCREASE/DECREASE buons to enter TIME. Press select to enter me and advance to next step.
16. Use INCREASE/DECREASE buons to enter SPEED. The knob can be used to adjust speed during infusion
process.
17. Press START to begin the INFUSE process.
